# Installation Steps
------------
## Composer
composer install

## Generate Key
php artisan key:generate

## Database Credentials
DB_HOST=localhost
DB_DATABASE=blog1
DB_USERNAME=test
DB_PASSWORD=12345

## Get Tables
php artisan migrate

## Run Db Seed
php artisan db:seed

## Run the project
php artisan serve
